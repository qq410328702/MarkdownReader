# Markdown Reader v0.0.1

首个公开发布版本！

## 功能特性

- 📖 支持 Markdown 文件阅读和预览
- 🎨 现代化的用户界面
- 🔍 内置搜索功能
- 📑 自动生成目录
- 🌓 支持主题切换
- 📤 支持导出为 HTML
- 🔗 自动关联 .md, .markdown, .mdown 文件

## 下载说明

### 安装版 (推荐)

**文件**: `MarkdownReader-v0.0.1-Setup.zip`

- 自动关联 Markdown 文件
- 创建开始菜单和桌面快捷方式
- 支持通过控制面板卸载

**安装步骤**:
1. 下载并解压 `MarkdownReader-v0.0.1-Setup.zip`
2. 右键点击 `install.cmd`
3. 选择"以管理员身份运行"
4. 按照提示完成安装

### 绿色便携版

**文件**: `MarkdownReader-v0.0.1-Portable.zip`

- 无需安装，解压即用
- 不修改系统设置
- 适合临时使用或 U 盘携带

**使用方法**:
1. 下载并解压 `MarkdownReader-v0.0.1-Portable.zip`
2. 运行 `MarkdownReader.exe`
3. 拖放 .md 文件到窗口打开

## 系统要求

- Windows 10 或更高版本
- .NET 8.0 运行时 (已包含)
- 约 150 MB 磁盘空间
